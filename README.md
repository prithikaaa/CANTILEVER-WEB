# Admin Dashboard

A modern React-based Admin Dashboard built to streamline administrative workflows through authentication, user management, analytics visualization, and task tracking.

## Features

### Authentication

* Secure login interface
* Protected dashboard access
* Session-based user flow

### Dashboard Analytics

* Overview cards displaying key metrics
* Interactive charts and visual insights
* Real-time data representation

### User Management (CRUD)

* Create new users
* View user information
* Update existing user records
* Delete users
* Search and filter functionality

### Kanban Board

* Task management using Kanban methodology
* Drag-and-drop task organization
* Multiple workflow stages:

  * To Do
  * In Progress
  * Completed

### API Integration

* REST API integration using Fetch API
* GraphQL support for efficient data fetching
* Dynamic data rendering

## Installation

Clone the repository:

```bash
git clone <repository-url>
```

Navigate to the project directory:

```bash
cd project-name
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm start
```

The application will run locally at:

```text
http://localhost:3000
```
